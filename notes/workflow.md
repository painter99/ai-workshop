### *AI Learning Approach (t3.chat)

Používám strukturovaný context prompt, který nastavuje AI jako **"Senior Mentor"**, ne code generator.

**Klíčové principy:**

1. **Socratic Method**  
   AI se ptá, nevydává hotová řešení. Musím problém vyřešit sám.

2. **No Code by Default**  
   Kód vidím **pouze** když explicitně vyžádám příkazem `ODHALIT`.  
   Jinak dostanu jen návod, otázky a koncepty.

3. **First Principles Focus**  
   Pochopení "proč" před "jak". Žádné copy-paste solutions.

4. **Curriculum Guard**  
   AI mě vrátí k základům, pokud skočím k pokročilým tématům předčasně.

5. **Error Categorization**  
   - Syntax errors = "mechanical failures" (rychlé opravy)
   - Logic errors = "process failures" (zastavení, analýza data flow)

**Proč tento přístup:**  
Brání "tutorial hell" — učím se **řešit problémy**, ne kopírovat řešení. AI je mentor, ne automat na kód.

**Kontrast:**
- ❌ Standard: "Write me a todo app" → copy-paste → nic jsem se nenaučil
- ✅ Můj přístup: "Proč tenhle loop nefunguje?" → AI: "Co se děje s indexem?" → sám hledám bug

**Výsledek:** Pomalejší progress, ale skutečné porozumění.
