# PHASE 0 — SETUP & WORKFLOW BASE (Week 1, 2–5h total)

## Objective
Build a stable local dev loop on Linux. No "fancy", just repeatable.

## What you DO
- Create a working folder for learning (e.g., `~/projects/python-workshop`)
- Install Python (system or pyenv) and verify `python3 --version`
- Create and use a venv:
  ```bash
  python3 -m venv .venv
  source .venv/bin/activate
  ```
- Install a package and import it (sanity check)

## Deliverables (in a repo OR local folder if repo is private)
- A minimal README.md with:
  - OS (Linux distro) & Python version
  - Venv workflow instructions
  - How to run a script

## Gate 0 (PASS/FAIL)
PASS if you can:
- Run `python3 file.py` from terminal
- Create+activate venv
- `pip install` works and import succeeds
